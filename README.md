# Resumate

Resumate — это современный веб-сайт для создания и редактирования профессиональных резюме онлайн. Платформа предлагает удобный интерфейс с шаблонами, автоматической версткой и экспортом в PDF.

Стэк: HTML & CSS & JS

# Запуск программы
Для того чтобы запустить конструктор Resumate нужно:
1) Нажать на зеленую кнопку Code
2) Скачать ZIP-файл и распаковать
3) Запустить main.html
